# DDOS-Gaurdian
Real-time DDoS detection &amp; prevention using supervised ML on network traffic patterns.
This project is a **B.Tech Final Year Research Work** focused on detecting and preventing **DDoS (Distributed Denial of Service) attacks** by analyzing network traffic patterns. Currently, this repository contains the **frontend part** of the system with a modern, responsive UI. The backend (ML model & real-time detection engine) will be integrated in the upcoming versions.

---

## 🚀 Features (Frontend Completed)
- ⚡ Responsive cybersecurity-themed UI built with **React + TailwindCSS**
- 🎨 Smooth animations using **Framer Motion**
- 📩 Functional **Contact Form** with form validation
- 👩‍💻 Team members & guide details showcased
- 🌑 Futuristic dark-mode design with neon highlights

---

## 🔧 Tech Stack
- **Frontend:** React, TailwindCSS, Framer Motion, Lucide Icons
- **Forms & Validation:** React Hook Form
- **Charts/Dashboard (Planned):** Chart.js / Recharts
- **Backend (Planned):** Python (Flask/Django) + ML models
